1. deploy git

  1. 使用git deploy的时候，格式问题，你直接在
    deploy:
    type: git |–>统一格式->|用户名| |->你的repository,”.git 必须添加”|
    repository: git@github.com:Notelzg/notelzg.github.com.git
  1. 每次需要输入用户名密密码解决：
    http://www.imooc.com/article/4433

2. about

about ，history，等文件夹的创建是在hexo根目录的source文件夹，不是主题文件夹下
可以使用命令直接创建

3. Disqus

question: 加入 Disqus 评论时也出现了『We were unable to load Disqus. If you are a moderator please see our troubleshooting guide』这个问题。
answer: 那是你的shortname 填写的是你的username ，你去disqus，点击设置按钮，选择下拉列表的里面的 Admin 选项，然后你看一下你的浏览器 显示的地址https://notelzg.disqus.com/admin/ 这个地址是我的地址，notelzg，就是我的shortname，我也是找了这个找了半天，填上正确的shortname之后肯定没有问题，你试一下
